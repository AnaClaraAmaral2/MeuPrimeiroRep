# Guia do GitHub
## 1° passo: Criar repositório. 
## 2° passo: Deixar público.
## 3° passo: Ir para code e copiar o link.
## 4° passo: ir para a explorador de arquivos e abrir a pasta que vai ficar o arquivo.
## 5° passo: Apertar botão doreito e abrir no Terminal.
## 6° passo: Escreva "git clone " cole o link que foi copiado. 
## 7° passo: Escreva o comando "cd" com o nome do reposiório e entre.
## 8° passo: Escreva o comando " code ." para abrir o vscode.
## 9° passo: Escreva o comando "git add .".
## 10° passo: Escreva o comando "git commit -m "xxxx"" para deixar uma mensagem sobre a mudança realizada.
## 11° passo: Escreva o comando "git push -u origin main" para empurrar da origem para o principal, lembrar de entrar na conta do git.
